# Traslation Web App 

# Technologies Used:-
    LangChain :- https://www.langchain.com

    OpenAI :- https://openai.com

    React Js :- https://react.dev


# Steps to install

    1- Run npm install in Project root directory.
    2- Add openAIApiKey to App.js.
    3- Run npm start.
